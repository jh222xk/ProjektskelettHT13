attr()

The attr expression is used to get the data in that selected HTML element.

Example

<a href="" title="Welcome to Web Development Reference"><a>


a:before {
   content: attr(title) ": ";
}

Browser support

  • Firefox 1.0
  • Chrome 2.0
  • Internet Explorer 8.0
  • Opera 9.0
  • Safari 3.1

border-radius

With the border-radius property can you easily create rounded corners in different elements.

Example

.rounded-corners {
    -moz-border-radius: 10px;
    -webkit-border-radius: 10px; 
    -o-border-radius: 10px; 
    border-radius: 10px;
}

This is a rounded box.

Browser support

  • Firefox 3.0 (Partially 2.0)
  • Chrome 4.0
  • Internet Explorer 9.0
  • Opera 10.5
  • Safari 3.1

box-shadow

With the box-shadow property can you easily implement drop shadows on elements both outer and inner.

Example

.drop-shadow {
    -moz-box-shadow: 5px 5px 5px #000;
    -webkit-box-shadow: 5px 5px 5px #000;
    box-shadow: 5px 5px 5px #000;
}

This box has a shadow.

Browser support

  • Firefox 3.5
  • Chrome 4.0
  • Internet Explorer 9.0
  • Opera 10.5
  • Safari 5.0 (Partially 3.1-4.0)

text-shadow

With the text-shadow property are you able to put shadow on your text of choice.

Example

.text-shadow {
    text-shadow: 2px 2px 2px #000;
}

This text has a shadow on it.

Browser support

  • Firefox 3.5
  • Chrome 4.0
  • Internet Explorer 10.0
  • Opera 9.5-9.6
  • Safari 4.0 (Partially 3.1-3.2)
